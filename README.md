# ecom-keyaccount-dashboard

Make a dashboard for a e-commerce key account manager in Lindt to keep overview of the price changes in the product "Lindt Exellence Extra Fine Creamy Chocolate" on three different Czechian stores, you can use simulated data for the last 7 days

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/ecom-keyaccount-dashboard.git
cd ecom-keyaccount-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
